Proyecto Angular: Galería de Imágenes

Un proyecto Angular es una aplicación de una sola página (SPA) construida utilizando el framework Angular. Este tipo de aplicaciones ofrecen una experiencia de usuario fluida y rápida al cargar la página una sola vez y actualizar solo los elementos necesarios en lugar de recargar la página completa. Angular facilita el desarrollo de aplicaciones dinámicas, escalables y fáciles de mantener.

En este proyecto, hemos creado una galería dinámica de imágenes utilizando Angular. La galería obtiene imágenes aleatorias desde una API externa y las muestra de manera elegante. Cada imagen representa un elemento de la lista con su respectiva información. Puedes ver las imágenes a continuación:
